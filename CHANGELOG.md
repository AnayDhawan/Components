All notable changes to this project are documented here. Format based on

## [0.4.0] - 2026-06-18

First public release. A Claude Code skill that fetches showpiece React and Tailwind
components live and adapts them to your brand tokens.

### Added
- Standalone repo with the skill payload (SKILL.md, components.json, references/) shipped
  alongside the community and meta files, so the README install command works out of the box.
- ATTRIBUTION.md crediting every upstream library with its license link; resolved the
  Aceternity license and flagged 21st.dev as per-component verify.
- README with tagline, quick start, "Why", rendered showpiece and fallback tables, usage
  examples, sources and licenses, supported tools, and community sections.
- SKILL.md troubleshooting section (registry, WebFetch, Playwright, graceful fallback).
- 18 showpiece effects and 12 plain fallbacks across 5 source libraries.
